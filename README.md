### **ESP-01 Relay Control via Website Button and Momentary Switch**

---
<br  />

**Features**
* Serial Monitor
* WiFi Connection
* Telnet Monitor
* OTA Updates
    * Firmware
    * Filesystem
* Website Hosting

**data>config.json**
1. The config.json file looks like this:

    ```
    {
        "deviceType" : "ESP01",
        "broadcastDeviceDetails" : 0,
        "ipAddress" : [192, 168, 1, 71],
        "accessPoint" : [
                            {
                                "ssid": "WiFi1",
                                "password": "12345"
                            },
                            {
                                "ssid": "WiFi2",
                                "password": "12345"
                            },
                            {
                                "ssid": "WiFi3",
                                "password": "12345"
                            }
                        ]
    }
    ```
